# WezTerm Command Cheatsheet (macOS)

## Tabs
| Action              | Shortcut           |
|---------------------|--------------------|
| New tab             | Cmd+T              |
| Close tab           | Cmd+W              |
| Next tab            | Ctrl+Tab           |
| Previous tab        | Ctrl+Shift+Tab     |
| Go to tab N         | Cmd+1 through Cmd+9|

## Panes (Splits)
| Action              | Shortcut                |
|---------------------|-------------------------|
| Split horizontal    | Cmd+Shift+D (or custom) |
| Split vertical      | Cmd+D (or custom)       |
| Navigate panes      | Ctrl+Shift+Arrow        |
| Close pane          | Cmd+W (closes focused)  |
| Zoom/unzoom pane    | Ctrl+Shift+Z            |
| Resize pane         | Ctrl+Shift+Arrow (hold) |

## Copy / Paste
| Action              | Shortcut           |
|---------------------|--------------------|
| Copy                | Cmd+C              |
| Paste               | Cmd+V              |
| Copy mode (vim-like)| Ctrl+Shift+X       |
| Quick select        | Ctrl+Shift+Space   |

## Scrolling
| Action              | Shortcut           |
|---------------------|--------------------|
| Scroll up           | Shift+PageUp       |
| Scroll down         | Shift+PageDown     |
| Scroll to top       | Shift+Home         |
| Scroll to bottom    | Shift+End          |

## Search
| Action              | Shortcut           |
|---------------------|--------------------|
| Search              | Cmd+F              |
| Next match          | Ctrl+Shift+N       |
| Previous match      | Ctrl+Shift+P       |

## Window / General
| Action              | Shortcut           |
|---------------------|--------------------|
| New window          | Cmd+N              |
| Command palette     | Cmd+Shift+P        |
| Toggle full screen  | Cmd+Enter           |
| Increase font size  | Cmd++              |
| Decrease font size  | Cmd+-              |
| Reset font size     | Cmd+0              |
| Reload config       | Cmd+R (or auto)    |
| Debug overlay       | Ctrl+Shift+L       |

## Tips
- **Command Palette** (Cmd+Shift+P) lets you search for any action by name
- Shortcuts are customizable in your `wezterm.lua` config
- WezTerm docs: https://wezfurlong.org/wezterm/config/keys.html
